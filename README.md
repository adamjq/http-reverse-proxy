# http-reverse-proxy

This repo contains a simple [reverse-proxy server](https://en.wikipedia.org/wiki/Reverse_proxy) written in Golang.

## Usage:

```bash

docker-compose up
```

Run [the requests](./requests.http) and inspect logs to see the service working.
